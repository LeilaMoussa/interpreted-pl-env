This ZIP file contains:
- Cover_Document.pdf: names and team
  info.

- interpreter.c: The interpreter code.
- assembler.c: the assembler code.

- MLcode1.txt: the simple ML program.
- MLcode2.txt: the medium complexity
  ML program.
- MLcode3.txt: the higher complexity 
  ML program.

- ML_Sample_Programs.pdf: details about
  the 3 ML programs and their expected
  output.

- ALcode1.txt: the simple AL program.
- ALcode2.txt: the medium complexity
  AL program.
- ALcode3.txt: the higher complexity
  AL program.

- AL_Sample_Programs.pdf: details about
  the 3 AL samples and the expected
  ML output files.

(Note that the 3 ML programs do not
do the same things as the 3 AL programs.)

- Assembler_Documentation.pdf
- Interpreter_Documentation.pdf

- Assembly_Language_Design_Document.pdf:
  design document for AL.
- Machine_Language_Design_Document.pdf
